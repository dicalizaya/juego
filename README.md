# 🎮 STORE.GG

![STORE.GG](img/juegos/LOGO.png)

## Plataforma de Comercio Electrónico para Videojuegos Digitales

STORE.GG es una aplicación web desarrollada para la gestión y comercialización de videojuegos digitales. El sistema permite a los usuarios explorar un catálogo de juegos, gestionar un carrito de compras, realizar compras y visualizar facturas, mientras que los administradores pueden gestionar videojuegos, categorías, plataformas, proveedores y usuarios.

---

## 🚀 Características Principales

### 👤 Usuarios
- Registro e inicio de sesión.
- Gestión de perfil.
- Exploración del catálogo de videojuegos.
- Búsqueda de juegos en tiempo real.
- Carrito de compras persistente.
- Confirmación de compra.
- Generación de factura.
- Historial de compras.

### 🛠️ Administración
- Gestión de videojuegos.
- Gestión de categorías.
- Gestión de plataformas.
- Gestión de proveedores.
- Gestión de usuarios.
- Control de acceso por roles.

---

## 🖥️ Tecnologías Utilizadas

- HTML5

- Fetch API
